Reporting Automation · 2023

48-hour reporting cycle, now 30 minutes.

End-to-end automation of a business-critical process that used to consume two working days and now runs before the morning coffee. Same-day decisions for the CFO and Chief Accountant.

Role Sole builder
Stack Python · SQL Server · Excel automation
Result 95% reduction in turnaround

The problem

A critical reporting process involving the Bookings and Billings view used by the CFO and Chief Accountant was taking two working days end-to-end. Multiple data sources, manual Excel work, complex SUMIFS, NETWORKDAYS and EOMONTH logic across four separate reporting views, cross-checks, emails. By the time decisions could be made, the numbers were already aging.

The approach

I rebuilt the pipeline end-to-end. A Python script pulls live from SQL Server, applies all the calculation logic natively in code rather than in nested Excel formulas, populates the Excel template staff already recognised, and delivers the finished report by email on a schedule.

The design constraint that mattered most: the finance team didn't want a new tool. They wanted the same output, faster. So the output format stayed identical — same template, same columns, same presentation. Everything that changed was upstream.

Why this was harder than it looks

The interesting part wasn't the Python. It was the domain — understanding exactly what "bookings vs billings" meant in this business, which edge cases mattered (order cancellations, partial shipments, multi-currency), and where the existing manual process was silently compensating for data issues upstream.

A pure technical implementation would have produced wrong numbers the finance team would have rejected. The real work was sitting with them, watching them work, asking why each step existed, and then codifying that.

Result

48 hours to 30 minutes. Same-day data-driven decisions became possible. More importantly, the finance team stopped losing two days a week to a process they didn't enjoy doing.

This wasn't an AI project. It's here because it's the work I'm most proud of — the unglamorous kind that actually moves the needle.
